# Simulation Data
This repository contains some simulation data in CSV format.

To to create a new CSV file, make sure that the first row includes parameter names.

Example below:

| count   | color  | active |
|---------|--------|--------|
| 2       | green  | true   |
| 8       | red    | false  |
| 3       | orange | true   |

One way to generate files is: https://www.mockaroo.com/
